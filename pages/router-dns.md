# [Router DNS]{style="color: green;"} configuration

🎯 Access homelab apps via homelab server name

* Enable router to resolve hostnames (default in OpenWRT)
* Set network domain name in router: `.lan`
* Set the homelab server hostname explicitly: `homelab`


Instead of http://localhost:8000, or http://192.168.33.30:8000,
<br>
you can use http://homelab.lan:8000 🎉
